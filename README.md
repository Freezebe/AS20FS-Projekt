# AS20FS-Projekt LinuxStartupDokumenterare
Ett startupscript för att dokumentera systemdetaljer och nätverksinformation

# Systemkrav
- Linux-baserat operativsystem
- Stöd för BASH programspråket

#Instruktioner
- Ge båda skripten tillåtelse att användas med chmod +x
- Lägg till Autorun.sh i listan för startupprogram genom Session och uppstart(Kali linux) eller liknande program, för att den ska starta vid login

# Syfte
Syftet med detta skript är att förenkla och automatisera system och nätverkskontroller för säkerhetsarbeten eller för privata säkerhetskontroller.

# Mål
Målet med detta skript är att det ska köras automatiskt vid inloggning och kunna läsa av systeminformation och nätverksdetaljer som är viktiga inom cybersäkerhet.

# Funktioner
- Operativsystemscheck
- Kernel setup check
- Intern och extern IP visning
- Port skanning
- Visar aktiva applikationer och tjänster
